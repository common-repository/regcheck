== Installation ==

Install Regcheck either via the WordPress.org plugin directory, or by uploading the files to your server.

Activate the plugin through the 'Plugins' menu in WordPress.

Go to Tools - Regcheck settings and set your desired options.

To create an API please register on this site  – regcheck.org.uk


== Shortcodes ==

Use these shortcodes to add check field on the page

[regcheck number="on"]
[regcheck vin="on"]
